Q. Difference between angular and react

Angular is typescript based fronted framework , developed by google
It helps us build a single page application (SPA)
SPA - where the page does not fully reload when navigation between different views.

Angular Tools  -
node js, npm (Node package manager), CLI(Command Line Interface)

CLI - Used for create and manage angular apps

IQ :-
1.How to create angular project
2.what is npm and ng

ng - is a initials of angular

Steps to create angular project 
1. check node version
2. create folder Angular
3. Open cmd
4. type command -> npm install -g @angular/cli -> press enter
5. type command -> ng new project_name -> press enter
6. after above command it will ask some question 
   first question press yes
   for CSS press yes
   for server-side rendering press no
   last question press none (enter)
7. project will be created in angular folder --> open in vs code
8. in new terminal -> to run code --> ng serve
9. follow the link the page will be displayed.
